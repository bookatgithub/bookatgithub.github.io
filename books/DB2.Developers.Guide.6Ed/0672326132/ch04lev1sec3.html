What Is a User-Defined Data Type?

A user-defined data type, or UDT for short, provides a mechanism for extending the type of data that can be stored in DB2 databases, and the way that the data is treated. The UDT, after defined and implemented, extends the functionality of DB2 by enabling users to specify the UDT in DDL CREATE TABLE statements, just like built-in DB2 data types.
