TSO Commands

The DB2 TSO commands are CLISTs that can be used to help compile and run DB2 programs or build utility JCL. The TSO commands are issued from a TSO session, either online using ISPF panels or in batch using the IKJEFT01 program. There are two TSO commands:

DSNH

Can be used to precompile, translate, compile, link, bind, and run DB2 application programs written in COBOL, Assembler, FORTRAN, PL/I, or C.

DSNU

Can be used to generate JCL for online DB2 utilities.
