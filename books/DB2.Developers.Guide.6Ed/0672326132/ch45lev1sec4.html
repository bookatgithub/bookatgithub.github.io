Accessing the Data Warehouse

After you design the data warehouse, you can use data access tools (also known as business intelligence tools) to access the data. You can use many types of data access tools, including the following:

  • GUI or Web-based database query tools

  • Complex report writers

  • OLAP tools that analyze data along dimensions

  • Data mining tools

  • CASE tools

  • Program generation tools

Most data warehouses deploy only the first three categories of data access tools for end-user querying and analysis. Additionally, data mining is gaining acceptance. Data mining is the practice of automatic and systematic analysis of data to find patterns and trends. The topic of data mining is beyond the scope of this book.
