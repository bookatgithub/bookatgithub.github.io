A.1 The Python Language Web Site

The single most important source of information is http://www.python.org.This web site is the focal point of the Python community. All Python software, documentation, and other information is available either on that web site directly or from locations listed on the web site. We encourage you to spend a fair bit of time exploring it, as it's quite large and comprehensive. See Figure A.1 for a snapshot of the web site's home page .
