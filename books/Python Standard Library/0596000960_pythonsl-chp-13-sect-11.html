13.11 The compileall Module

The compileall module (see Example 13-11) contains functions to compile all Python scripts in a given directory (or along the Python path) to byte code. It can also be used as a script (on Unix platforms, it's automatically run when Python is installed).

Example 13-11. Using the compileall Module to Compile All Scripts in a Directory
File: compileall-example-1.py

import compileall

print "This may take a while!"

compileall.compile_dir(".", force=1)
