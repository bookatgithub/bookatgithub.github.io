13.9 The sre Module

(Implementation) The sre module, used in Example 13-9, is a low-level implementation module for the 2.0 re module. There's usually no need to use this module directly (and code using it may stop working in future releases).

Example 13-9. Using the sre Module
File: sre-example-1.py

import sre

text = "The Bookshop Sketch"

# a single character
m = sre.match(".", text)
if m: print repr("."), "=>", repr(m.group(0))

# and so on, for all 're' examples...
