7.11 The gopherlib Module

The gopherlib module contains a gopher client implementation, shown in Example 7-25.

Example 7-25. Using the gopherlib Module
File: gopherlib-example-1.py

import gopherlib

host = "gopher.spam.egg"

f = gopherlib.send_selector("1/", host)
for item in gopherlib.get_directory(f):
    print item
