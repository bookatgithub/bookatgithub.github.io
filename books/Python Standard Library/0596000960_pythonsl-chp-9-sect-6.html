9.6 The sunau Module

The sunau module in Example 9-5 reads and writes Sun AU audio files.

Example 9-5. Using the sunau Module
File: sunau-example-1.py

import sunau

w = sunau.open("samples/sample.au", "r")

if w.getnchannels() == 1:
    print "mono,",
else:
    print "stereo,",

print w.getsampwidth()*8, "bits,",
print w.getframerate(), "Hz sampling rate"
