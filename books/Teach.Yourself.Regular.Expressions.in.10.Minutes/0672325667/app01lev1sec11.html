MySQL

MySQL is a popular open source database, and MySQL ventures where no other databases have yet to venture by providing regular expression support as a way to perform database searches.

Regular expression support in MySQL is accessed in WHERE clauses in the following format:


REGEXP "expression"

Note

A complete MySQL statement using a regular expression would use syntax like this: SELECT * FROM table WHERE REGEXP "pattern".


MySQL regular expression support is useful and powerful, but it is not a full regular expression implementation:

  • Only search support is provided; there is no support for replace operations.

  • Searches are not case sensitive. To perform case-sensitive searches, use the BINARY keyword (between REGEXP and the expression itself).

  • Use [[:<:]] to match the start of a word and [[:>:]] to match the end of a word.

  • Lookaround is not supported.

  • Embedded conditions are not supported.

  • Octal character searching is not supported.

  • \a, \b, \e, \f, and \v are not supported.

  • Backreferences are not supported.
