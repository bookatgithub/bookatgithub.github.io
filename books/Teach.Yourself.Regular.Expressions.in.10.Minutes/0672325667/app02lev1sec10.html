HTML Comments

Comments in HTML pages are placed between <!-- and --> tags (use at least two hyphens, although more are allowed). Being able to locate all comments is useful when browsing (and debugging) Web pages.


<!-- Start of page -->

<HTML>

<!-- Start of head -->

<HEAD>

<TITLE>My Title</TITLE> <!-- Page title -->

</HEAD>

<!-- Body -->

<BODY>



<!-{2,}.*?-{2,}>



<!-- Start of page -->

<HTML>

<!-- Start of head -->

<HEAD>

<TITLE>My Title</TITLE> <!-- Page title -->

</HEAD>

<!-- Body -->

<BODY>

<!-{2,} matches the start of the comment, <! followed by two or more hyphens. .*? matches the comment body (not greedy). -{2,}> matches the end of the comment.

Note

This regular expression matches two or more hyphens and can thus be used to find CFML comments, too (which are identified by three hyphens). However, the pattern does not attempt to match the number of hyphens at the comment's start and close (potentially a useful enhancement in finding mismatched comments).
