Each function returns true if its argument is one of the standard blank charactersspace and horizontal tab (i.e., ' ' and '\t' for isblank, L' ' and L'\t' for iswblankor if it is one of a locale-specific set of characters that are used to separate words in a line of text, in which case, isspace or iswspace, respectively, must also return true.
